## Setup Instructions

### 1. Supabase Database Setup

1. Create a new project at [Supabase](https://supabase.com)
2. Go to the SQL Editor and run the following SQL to create the resumes table:

```sql
-- Create resumes table
CREATE TABLE resumes (
    id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT TIMEZONE('utc'::text, NOW()),
    full_name TEXT NOT NULL,
    university TEXT,
    skills TEXT[],
    experience_years TEXT,
    experience_details TEXT,
    resume_file_url TEXT
);

-- Enable Row Level Security (optional)
ALTER TABLE resumes ENABLE ROW LEVEL SECURITY;

-- Create policy for insert (anonymous access)
CREATE POLICY "Allow anonymous inserts" ON resumes
    FOR INSERT TO anon, authenticated
    WITH CHECK (true);
```

3. Get your Supabase credentials:
   - Go to Project Settings → API
   - Copy the **Project URL** and **anon public** key

### 2. Configure PHP Backend

Edit the file `php/submit_resume.php` and replace the placeholder values:

```php
$supabase_url = 'YOUR_SUPABASE_URL';        // e.g., 'https://xyzcompany.supabase.co'
$supabase_key = 'YOUR_SUPABASE_ANON_KEY';   // e.g., 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...'
```

### 3. Deploy

Upload all files to your PHP-enabled web server:

```
ascenda/
├── index.html
├── submit-resume.html
├── css/
│   └── styles.css
├── php/
│   └── submit_resume.php
└── uploads/
```
